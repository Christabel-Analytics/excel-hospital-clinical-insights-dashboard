# Hospital Clinical Insights Dashboard

## Project Overview

This project analyzes over 55,000 patient records containing clinical, demographic, and hospital data using Microsoft Excel. The analysis aims to identify patient trends, healthcare utilization patterns, billing performance, and the prevalence of medical conditions through an interactive dashboard.

## Project Objectives

- Analyze patient demographics and healthcare utilization.
- Examine billing performance across medical conditions.
- Identify patient distribution trends.
- Compare hospital contributions to total billing.
- Develop an interactive dashboard to support healthcare decision-making.


## Tools Used

- Microsoft Excel
- formulas
- Pivot Tables
- Pivot Charts
- Slicers

## Data Cleaning & Preparation

- Standardized data formatting.
- Removed duplicate records.
- Handled missing values.
- Created helper columns (Month, Year, and Age Group).
- Structured billing data for analysis and visualization.

## Dashboard Features

The dashboard includes interactive slicers that allow users to filter the analysis by:

- Admission Type
- Month
- Gender


## Key Performance Indicators (KPIs)

- *Total Records: 55,000+
- *Unique Patients: 18,069
- *Total Billing Amount: £460,889,706
- *Abnormal Test Results: 34%


## Key Insights

- Patient distribution is higher among older age groups across both genders, indicating greater healthcare utilization among older populations.
- Diabetes accounts for the highest billing (16.8%); however, billing is relatively evenly distributed across all medical conditions.
- The top 10 hospitals contribute only 2.8% of total billing, indicating that healthcare expenditure is broadly distributed across hospitals within this dataset.
- Patient volume increased around 2020 before declining in 2024, which may reflect changes in healthcare utilization or reporting.
- Patient distribution across medical conditions is relatively balanced, with only slight variations.


## Recommendations

- Prioritize geriatric-focused care and strengthen chronic disease management (especially diabetes) to address higher utilization among older age groups.

- Improve coordination across multiple healthcare facilities by standardizing care pathways and cost tracking, given the highly decentralized billing patterns to improve visibility, comparability, and overall system efficiency in resource allocation.

-Investigate the decline in patient volume in 2024 to identify potential gaps in access, reporting, or service utilization.

-Follow up on the 34% abnormal test results with further checks and targeted care to improve early detection and treatment.

-Review the high total billing and many hospitals involved to understand what is driving costs and improve how resources are allocated.


## Conclusion

This analysis shows that, within this dataset, healthcare utilization and billing are broadly distributed across hospitals and medical conditions. The findings highlight the importance of system-wide planning and evidence-based resource allocation.
